# 프론트엔드 개발자를 위한 연결 정보

## 🌐 백엔드 서버 주소

**⚠️ IP 주소는 환경변수로 관리합니다. 하드코딩하지 마세요!**

백엔드 담당자에게 현재 로컬 IP 주소를 확인하세요:
```bash
# Mac
ipconfig getifaddr en0

# Windows
ipconfig

# Linux
hostname -I
```

## 📋 연결 방법

### 1. 같은 WiFi 네트워크에 연결
백엔드 서버를 실행한 컴퓨터와 **같은 WiFi**에 연결되어 있어야 합니다.

### 2. 백엔드 서버 실행 확인
백엔드 담당자가 서버를 실행했는지 확인:
```bash
# 백엔드 담당자에게 요청
# 서버 실행: npm start 또는 node backend/server.js
```

### 3. 프론트엔드에서 API 호출 (환경변수 사용)

**방법 1: config.js 사용 (권장)**
```html
<!-- HTML 파일의 <head> 섹션에 추가 -->
<script>
  // 백엔드 담당자가 알려준 로컬 IP 주소
  window.API_BASE_URL = 'http://<로컬IP>:3000';
</script>
<script src="config.js"></script>
```

```javascript
// 브랜드 목록 조회 예시
fetch(window.API_CONFIG.API_ENDPOINTS.brands)
  .then(res => res.json())
  .then(data => {
    console.log('브랜드 목록:', data.brands);
  });
```

**방법 2: 환경변수 파일 사용**
```bash
# .env.local 파일 생성 (Git에 커밋하지 않음)
REACT_APP_API_URL=http://<로컬IP>:3000
```

## ✅ 연결 테스트

브라우저에서 직접 접속하여 테스트:
```
http://<로컬IP>:3000/health
```

정상 응답:
```json
{
  "status": "ok",
  "timestamp": "2025-01-31T..."
}
```

## ⚠️ 주의사항

1. **IP 주소 변경 가능성**: WiFi를 재연결하거나 네트워크가 변경되면 IP 주소가 바뀔 수 있습니다.
   - IP가 변경되면 백엔드 담당자에게 다시 확인 요청하세요.

2. **방화벽 설정**: 백엔드 서버 컴퓨터의 방화벽에서 포트 3000이 열려있어야 합니다.

3. **절대 경로 사용**: 프론트엔드에서 API 호출 시 반드시 절대 경로를 사용하세요.
   ```javascript
   // ❌ 잘못된 예시
   fetch('/api/brands')
   
   // ✅ 올바른 예시 (환경변수 사용)
   fetch(`${window.API_CONFIG.API_BASE_URL}/api/brands`)
   ```
