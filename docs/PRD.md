# Product Requirements Document (PRD)

## 제품명

**StartSmart Cafe Franchise**
AI 기반 프랜차이즈 카페 창업 현실 검증 시뮬레이터 (MVP / 해커톤 구현 버전)

---

## 0. 문서 목적

본 PRD는 프랜차이즈 카페 창업 의사결정을
**막연한 기대나 감각적 판단에서 벗어나, 수치·현장·판단을 결합한 실행 가능한 결정으로 전환**하기 위한
AI 기반 시뮬레이션 서비스의 요구사항을 정의한다.

StartSmart는

* 기대 판매량과 목표 판매량 간의 괴리를 드러내고
* **36개월 회수 기준**을 중심으로 구조적 위험을 판정하며
* 데이터·지도·이미지·AI 판단을 결합한 **AI 네이티브 창업 의사결정 도구**를 지향한다.

---

## 1. 제품 정의

### 1.1 한 줄 정의

> **서울 내 특정 입지(지도 핀)에서, 선택한 프랜차이즈 카페 브랜드가**
> **사용자의 조건(월세·평수·투자금·점주 근무 여부) 하에서**
> **얼마나 생존하고, 언제 회수 가능하며, 월 순이익이 얼마인지 계산하고**
> **AI가 구조적 리스크와 개선 가능성까지 판단해주는 창업 현실 검증 시뮬레이터**

---

### 1.2 핵심 가치

* **구체화**: 창업 조건을 실행 가능한 숫자(회수 개월, 순이익, 필요 판매량)로 전환
* **현실 검증**: 상권·비용·물리적 입지 조건이 목표 판매량을 허용하는지 검증
* **판단 보조**: 계산 결과를 AI가 컨설턴트 관점에서 해석·설명
* **재사용**: 조건을 변경하며 반복 시뮬레이션 → 최종 창업 결정까지 지원

---

## 2. MVP 범위 (확정)

### 2.1 대상 및 범위

* 업종: 프랜차이즈 카페 단일
* 브랜드: 12개 (정적 데이터셋)
* 지역: 서울 (초기 데모는 강남구로 제한 가능)
* 상권 분석: 지도 핀 기반 반경 분석

  * 300m / 500m / 1km
* 분석 레이어:

  * 재무 시뮬레이션
  * 상권 경쟁도
  * **AI 로드뷰 기반 물리 리스크 분석**
  * **AI 리스크 해석 & 개선 제안**
* 산출물:

  * 결과 대시보드
  * PDF 리포트

### 2.2 판정 기준

* 점수(0~100)
* 신호등(🟢🟡🔴)
* 예상 생존 개월 수
* 하드컷 규칙:

  * 회수 기간 ≥ 36개월 → 🔴 위험
  * 월 순이익 ≤ 0 → 🔴 위험

---

## 3. 사용자 흐름 (User Workflow)

### 3.1 핵심 사용자 루프 (데모 필수)

1. 프랜차이즈 브랜드 선택
2. 지도에서 입지 핀 클릭
3. 입력값 세팅

   * 초기 투자금
   * 월세
   * 평수
   * 점주 근무 여부 (Y/N)
4. AI 판매량 시나리오 제안 (보수 / 기대 / 낙관)
5. 사용자가 목표 판매량 입력
6. 계산 결과 생성

   * 회수 개월 / 순이익 / 민감도
7. **AI 로드뷰 분석관 실행 (현장 물리 리스크)**
8. **AI 리스크 분석 & 개선 제안 자동 생성**
9. 결과 대시보드 확인
10. 조건 수정 → 재계산 → 결과 비교
11. PDF 리포트 출력 및 공유

---

## 4. 기능 리스트 (Feature List)

### 4.1 기능 모듈 트리

* **F1.** 브랜드 선택 / 브랜드 데이터 로딩
* **F2.** 지도 핀 기반 상권 분석 (300m / 500m / 1km)
* **F2-1. AI 로드뷰 분석관 (Physical Risk Analyzer)**

  * 로드뷰 이미지 기반 현장 물리 리스크 인식
  * 간판 가림 / 급경사 / 층위 추정 / 보행 가시성 평가
* **F3.** 입력 폼 (기본값 + 고급 입력)
* **F4.** 판매량 AI 추론 (현실적 달성 가능 구간 제안)
* **F5.** 손익 계산 엔진 (회수 개월 / 순이익 / 민감도)
* **F6.** 점수 / 신호등 / 생존 개월 판단 엔진
* **F7.** 리스크 문장 생성 (AI 판단 기반)
* **F7-1. AI 리스크 분석 & 개선 제안 엔진 (Consulting AI Layer)**
* **F7-2. 실시간 경쟁 환경 해석 AI (Competitive Intelligence AI)**
* **F8.** 대시보드 시각화
* **F9.** PDF 리포트 내보내기
* **F10.** 시뮬레이션 저장 / 불러오기
* **F11.** 배포 / 데모 스위치
* **F12. (Phase2)** AI 심층 상담관 (Post-Analysis Advisor)

---

## 5. AI 핵심 역할 정의 (AI Necessity)

StartSmart에서 AI는 보조 도구가 아닌 **판단 주체**로 동작한다.

### 5.1 AI의 역할

* 계산 결과 해석 및 구조적 리스크 식별
* 목표 판매량의 현실성 판단
* 경쟁 환경에서의 생존 전략 추론
* 물리적 입지 리스크(로드뷰) 인식
* 개선 레버(무엇을 바꾸면 효과적인지) 제안

### 5.2 AI 없이는 불가능한 영역

* 이미지 기반 물리 리스크 인식
* 다변수 결과의 종합 판단 언어화
* 경쟁 구도의 정성적 해석

---

## 6. 성공 기준 (Definition of Done)

* 입력 → 결과 → 수정 → 재결과 루프 정상 동작
* 재무 / 상권 / 물리 리스크가 결과에 반영
* 회수 개월 / 순이익 / 민감도 계산 정확
* AI 리스크 분석 및 개선 제안 자동 생성
* PDF 리포트 생성 성공
* 데모 시나리오 완주 가능

---

## 7. 최종 한 줄 요약 (Pitch)

> **StartSmart는 프랜차이즈 카페 창업을**
> **‘될 것 같은지’가 아니라**
> **‘수치·현장·구조상 가능한지’로 판단하게 만드는 AI 의사결정 시뮬레이터다.**
