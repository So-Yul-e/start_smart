### 동영 — Frontend 역할 배정 이유

동영님 사양(i3 + 8GB + GPU 없음)은 팀 내 최저이지만, **Frontend 역할이 가장 적합**합니다:

**가능한 이유:**
- 프론트엔드는 **브라우저 + 에디터만** 있으면 됨
- Node.js 서버를 직접 돌릴 필요 없음 (소율이 서버 담당)
- API 키 필요 없음
- HTML/CSS/JS 파일만 수정
- jsPDF는 클라이언트 사이드에서 동작 (서버 부하 없음)

**주의할 점:**
- VS Code가 무거울 수 있음 → **메모장++ 또는 Sublime Text** 사용 추천 (가벼움)
- 브라우저 탭 많이 열지 않기 (RAM 8GB 한계)
- 로컬 테스트는 HTML 파일 직접 열기로 (Live Server도 괜찮음)
- 전체 동작 테스트는 **소율 서버에 접속**해서 확인 (본인 PC 부담 없음)
- 지도 API는 백엔드에서 처리하므로 프론트엔드 부담 없음

**8GB로 힘들면:**
- Replit에서 브라우저로 코딩 (로컬 부담 제로)
- 크레딧 $25 있으므로 충분

## 이 프로젝트에 로컬 사양이 중요한가?

**결론: 거의 안 중요합니다.**

모든 AI 처리는 클라우드 API 호출로 이루어집니다:

| 작업 | 처리 위치 | 로컬 부하 |
|------|-----------|-----------|
| 로드뷰 이미지 분석 | Gemini Vision API (클라우드) | 거의 없음 |
| 판매량 추론 / 리스크 분석 | Claude API (클라우드) | 거의 없음 |
| 손익 계산 | 로컬 (순수 계산) | 매우 가벼움 |
| 점수/신호등 판단 | 로컬 (순수 계산) | 매우 가벼움 |
| 상권 분석 | 지도 API (클라우드) | 거의 없음 |
| Express 서버 | 로컬 | 매우 가벼움 (Node.js) |
| 프론트엔드 | 브라우저 | 매우 가벼움 (HTML/CSS/JS) |

**로컬에서 필요한 것**: 코드 에디터 + 브라우저 → 어떤 사양이든 충분

## GCP 없이 돌아가나?

**네, 완전히 돌아갑니다.**

이 프로젝트는 로컬 `node backend/server.js`만 실행하면 됩니다.
- Express 서버: 로컬 (포트 3000)
- AI 처리: 전부 외부 API 호출 (인터넷만 있으면 됨)
- 계산 엔진: 로컬 (순수 JavaScript 계산)
- 상권 분석: 지도 API 호출 (인터넷만 있으면 됨)
- 파일 저장: 로컬 (필요시)
- DB: 메모리 저장소 (해커톤용)

소율 노트북(M4 Pro 48GB)이면 서버 + 브라우저 + 에디터 동시에 켜도 여유 넘칩니다.

## 저사양 팀원을 위한 개발 환경 대안

### 1. Replit (추천 — 특히 동영, 민균)
- 브라우저에서 코딩 + 실행 가능
- 팀 크레딧 $125 있음
- 로컬 사양 무관

### 2. GitHub Codespaces
- 브라우저에서 VS Code 사용
- 무료 60시간/월
- 4코어 + 16GB RAM 클라우드 환경

### 3. 소율 노트북에서 통합 서버 실행
- 소율이 서버를 띄우고, 같은 네트워크에서 다른 팀원이 접속
- `http://<소율IP>:3000`으로 테스트 가능
- 저사양 팀원은 코드만 작성 → Git push → 소율이 pull해서 통합 테스트

## 성능 최적화 팁

### 저사양 PC (동영, 민균)
1. **에디터 최적화**
   - VS Code 확장 프로그램 최소화
   - 가벼운 에디터 사용 (Sublime Text, 메모장++)
   
2. **브라우저 최적화**
   - 불필요한 탭 닫기
   - 확장 프로그램 비활성화
   - 개발자 도구는 필요할 때만 열기

3. **시스템 최적화**
   - 백그라운드 앱 종료
   - 불필요한 서비스 중지
   - RAM 정리 (Mac: Activity Monitor, Windows: Task Manager)

### 중간 사양 PC (다효)
- 일반적인 개발 환경으로 충분
- Node.js + 에디터 + 브라우저 동시 실행 가능

### 고사양 PC (대성, 소율)
- 모든 도구 동시 실행 가능
- 통합 테스트 및 디버깅 여유롭게 진행
