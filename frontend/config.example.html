프론트엔드 API 설정 방법

방법 1: HTML에서 직접 설정 (권장)

각 HTML 파일의 <head> 섹션에 다음 스크립트를 추가하세요:

<script>
  // 백엔드 담당자가 알려준 로컬 IP 주소 (환경변수로 관리)
  window.API_BASE_URL = 'http://<로컬IP>:3000';
</script>
<script src="config.js"></script>
<script src="your-app.js"></script>

방법 2: 환경변수 파일 사용

프로젝트 루트에 .env.local 파일을 만들고:

REACT_APP_API_URL=http://<로컬IP>:3000

빌드 도구에서 환경변수를 주입하도록 설정하세요.

사용 예시

// config.js를 로드한 후
fetch(API_ENDPOINTS.brands)
  .then(res => res.json())
  .then(data => {
    console.log('브랜드 목록:', data.brands);
  });
