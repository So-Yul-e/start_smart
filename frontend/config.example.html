프론트엔드 API 설정 방법

방법 1: 기본 설정 (로컬 개발 환경)

기본적으로 config.js는 http://localhost:3000을 사용합니다.

별도 설정 없이 바로 사용 가능합니다:

<script src="config.js"></script>
<script src="your-app.js"></script>

방법 2: 다른 환경에서 사용 (선택사항)

다른 IP나 프로덕션 환경에서 사용하려면 HTML에서 설정:

<script>
  // 다른 환경에서만 설정 (로컬 개발 환경에서는 불필요)
  window.API_BASE_URL = 'http://<다른IP>:3000';
</script>
<script src="config.js"></script>

방법 3: 환경변수 파일 사용 (빌드 도구 사용 시)

프로젝트 루트에 .env.local 파일을 만들고:

REACT_APP_API_URL=http://localhost:3000

빌드 도구에서 환경변수를 주입하도록 설정하세요.

사용 예시

// config.js를 로드한 후
fetch(API_ENDPOINTS.brands)
  .then(res => res.json())
  .then(data => {
    console.log('브랜드 목록:', data.brands);
  });
