## 💸 9. 폐업(Exit) 비용 상세 (Exit Cost Breakdown)

폐업은 "그만두는 선택"이 아니라 **추가 비용이 발생하는 이벤트**입니다. 아래는 36개월에 폐업한다고 가정했을 때의 비용 구조입니다.

|------|------:|
| 가맹 위약금 | 0만원 |
| 철거/원상복구 | 2,500만원 |
| 인테리어/설비 손실(비회수) | 4,900만원 |
| 권리금 회수(감액) | -3,000만원 |
| **Exit Cost 합계** | **4,400만원** |

| **최종 총손실** | **4,400만원** |

### 비용 구조 해석

- **위약금**: 계약 기간이 만료되어 위약금이 없습니다.
- **철거/원복**: 인테리어 철거 및 원상복구 비용입니다.
- **인테리어 손실**: 초기 투자금 중 인테리어/설비 비중의 비회수 부분입니다.
