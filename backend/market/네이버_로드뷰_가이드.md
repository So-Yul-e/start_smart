# 네이버 로드뷰 API 가이드

## ⚠️ 중요: 네이버는 로드뷰 API를 제공하지 않습니다

네이버 지도 API는 **로드뷰 기능을 제공하지 않습니다**. 네이버가 제공하는 것은 **Static Map API**뿐이며, 이는 일반 지도 이미지만 생성합니다.

---

## 네이버 Static Map API

### 제공 기능
- 일반 지도 이미지 (로드뷰 아님)
- 마커 표시
- 줌 레벨 조정
- 지도 유형 선택 (일반/위성/하이브리드)

### 서버 사이드 호출
- ✅ REST API로 서버 사이드에서 직접 호출 가능
- ✅ Static Map 이미지 URL 생성 가능

### 제한사항
- ❌ 로드뷰 이미지 제공 불가
- ❌ 실제 거리 뷰 이미지 없음
- ❌ AI 로드뷰 분석에는 부적합

---

## 현재 구현 (`roadviewApi.js`)

```javascript
function getNaverRoadviewUrl(location, options = {}) {
  // 네이버 Static Map API 사용
  // ⚠️ 이는 지도 이미지일 뿐, 로드뷰가 아닙니다.
  const url = `https://naveropenapi.apigw.ntruss.com/map-static/v2/raster?w=${width}&h=${height}&center=${location.lng},${location.lat}&level=3&markers=...`;
  return url;
}
```

**반환값**:
- 일반 지도 이미지 URL
- 로드뷰 이미지가 아님
- AI 로드뷰 분석에는 부적합

## 로드뷰 API 비교

| 제공자 | 로드뷰 API | 서버 사이드 호출 | AI 분석 적합성 |
|--------|-----------|----------------|---------------|
| **Google** | ✅ Street View Static API | ✅ 가능 | ✅ **가장 적합** |
| **카카오** | ✅ JavaScript API만 | ❌ 불가능 | ❌ 프론트엔드 전용 |
| **네이버** | ❌ 제공 안 함 | ❌ 해당 없음 | ❌ 지도 이미지만 제공 |

## 권장 사항

### AI 로드뷰 분석이 목적인 경우

**Google Street View Static API 사용을 강력히 권장합니다**:

1. ✅ 서버 사이드에서 직접 호출 가능
2. ✅ 실제 로드뷰 이미지 제공
3. ✅ Gemini Vision API로 분석 가능
4. ✅ 입지 리스크 분석에 적합

**설정 방법**:
1. `.env` 파일에 `GOOGLE_MAPS_API_KEY` 설정
2. [Google Cloud Console](https://console.cloud.google.com/)에서 **Street View Static API** 활성화
3. `getGoogleStreetViewUrl()` 함수 사용

### 사용자에게 로드뷰 보여주기

**프론트엔드에서 카카오 로드뷰 JavaScript API 사용**:
- 카카오 로드뷰는 JavaScript API로만 제공
- 프론트엔드에서 사용자에게 로드뷰 표시
- 서버 사이드 분석과는 별개로 동작

---

## 결론

**네이버는 로드뷰 API를 제공하지 않으므로**, AI 로드뷰 분석을 위해서는:

1. **Google Street View Static API** 사용 (권장)
2. 프론트엔드에서 카카오 로드뷰 JavaScript API 사용 (사용자 표시용)

네이버 Static Map은 로드뷰 대안으로 사용할 수 없습니다.
